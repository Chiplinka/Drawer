# Drawer
Программа для рисования, сделанная на PyQt5.

# Реализованы :
- **3 Вида пера**

  - Кисть
  - Аэрозоль
  - Каллиграфическая кисть
  
- **Выбор размера пера**
   - 1px
   - 3px
   - 5px
   - 8px
 
- **Выбор цвета пера**
   - Черный
   - Красный
   - Синий
   - Зеленый
   - Желтый
   - Выбор цвета пера с помощью диалогового окна
- **Функции**

   - Сохранить
   - Очистить холст
   - Нарисовать эллипс
   - Нарисовать Прямоугольник
