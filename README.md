# ProjectMain
Программа для рисования, сделанная на PyQt5.

# Реализованы :
- **3 Вида пера**

  - Кисть
  - Аэрозоль
  - Каллиграфическая кисть
  
- **Функции**

   - Сохранения
   - Очищения хоста
   
- **Выбор размера пера**
   - 1px
   - 3px
   - 5px
   - 8px
 
- **Выбор цвета пера**
   - Черный
   - Красный
   - Синий
   - Зеленый
   - Желтый
- **А так же выбор цвета с помощью диалогового окна**
